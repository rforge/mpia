Interface ist ähnlich einer abstrakten Klasse,
allerdings ohne Implementierung!
Ein Interface soll eine einheitliche Schnittstelle
generieren!
In einem Interface werden die Methodenparameter (name, Übergabewerte, Rückgabewerte)
nur Signaturen und gegebenenfalls Variablen (sind implizit
static und final) definiert! Die Methoden
müssen in der Klasse die das Interface
verwendet (mit Befehl "implements name") genauer
spezifiziert werden.
Befehl zur Generierung: interface name
