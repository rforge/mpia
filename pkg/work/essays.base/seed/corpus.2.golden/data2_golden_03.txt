Es gibt keine eindeutige Definition von elektronischen Katalogen
Sie umfassen im wesentlichen Produktlistung, Preise und ermöglichen Transaktionen 

Client Request -> Web Server (Http-Server) <-> CGI-Script  -> SQL-Abfrage -> DBMS 
                                                              Common Gateway Interface -> Client-Server

Häufigste E-Commerce Anwendung 

Funktionweise Kunden:
virtueller Warenkorbfunktionalität (gekaufte Produkte werden dort abgelegt)
Auftragstatus der Bestellung kann abgefragt werden (Shipment Tracking)
Auftragsbestätigung mittels E-mail
Suchfunktion nach Produkt, Produktgruppe, Preis
Programme zur Community Building (Bsp. Amazon -> Buchklubs, reviewing)
Darstellung in verschiedenen Sprachen

Funktionsweise von Administrator:
Kundenregistrierung - durch Kunden selbst
Erstellung von Kundenprofile (Tracking, adaptive Portale, Indiviualisierung, 1 to 1 Marketing)
Zugriffsstatistik -> welche Seiten werden oft angewählt
Entwicklung von Entscheidungsbäumen (collaborative Filtering) und Rabattentscheidung -> automatisch
