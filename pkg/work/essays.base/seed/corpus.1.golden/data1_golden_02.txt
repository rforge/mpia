die Variablen nachname und angemeldet sind private deklariert; d.h. sie können nur aus der eigene Klasse angesprochen werden; um sie zu manipulieren oder auszulesen werden in derselben Klasse Methoden implementiert (diese müssen als public deklariert werden, damit man sie von außerhalb d. Klasse aufrufen kann), die auf diese Variablen zugreifen, man spricht auch von Datenkapselung (die Daten sind von den Methoden umschlossen); Information Hiding bietet die Vorteile daß die Implementierung d. Daten verändert werden kann ohne daß Code außerhalb d. Klasse geändert werden muß; außerdem wird d. Zugriff auf Variablen eingeschränkt um unerwünschte Datenmanipulation zu verhindern.
