Information Hiding ermöglicht es, den direkten Zugriff auf Variablen zu verhindern. bietet gewisse Sicherheit. Zum Beispiel:
public class Pruefung {
private int node;
public void setNote(int note) {
this.note = note;
}
public int getNote () {
return note;
}
ermöglicht "das Setzen" d. Variable note auf einen best. Wert. note wird ausgelesen
... somit wird nie direkt auf note zugegriffen, sondern nur über d. Methoden getNote bzw setNote
